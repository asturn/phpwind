(一) 更新版本：phpwind 8.7beta (0817)
(二) 适用版本：phpwind 8.7beta (0817)

(三) 更新步骤:

     第一步 确定站点的程序版本，右键查看网页源码可以查看版本，是0802还是0817。下载对应的升级包

     第二步 更新之前，建议先将站点关闭，并做好备份以备万一

     第三步 确定你站点的文件名、后台登陆文件、附件目录等是否有所改动
            首页文件查看路径：【后台】->【论坛模式】->【界面设置】->【首页】
    	    前台注册文件、后台登陆文件查看路径：【后台】->【全局】->【安全优化】->【安全控制】
    	    附件目录查看路径：【后台】->【全局】->【安全与优化】->【动态目录】

     第四步 将下载的优化包解压。检查得到的upload文件夹。如果第3步文件有改动，则将upload中的对应文件改成你站点的相应文件或目录名

     第五步 上传upload下的所有文件及目录，以覆盖原站点程序
      
     第六步 上传up87betato87.php，运行此文件

     第七步 站点优化完毕，重新访问你的站点，看看效果吧


     
（四）如果在使用中遇到问题,你可以
(1)前往官方FAQ平台：http://faq.phpwind.net/ 查找相关信息
(2)前往phpwind讨论区 http://www.phpwind.net/thread-htm-fid-2.html 发帖询问，工作日官方技术人员会提供解答
