安装详细流程

第一步：确认系统

请确认您空间或本机的操作系统，确认已经安装PHP+Mysql运行环境。否则，请参照【系统环境说明】。


第二步：上传文件

解压安装包，并将upload目录下的所有文件拷贝到根目录。
安装包文件说明：index.htm为安装相关说明；docs目录为用户使用相关文档及最新版的功能更新说明；upload目录为程序文件。
注意：如果您使用的是Unix、Linux等非Windows性质的空间、服务器，上传文件时，请确认以下文件属性为777。（上传失败请参见）
  • attachment
  • attachment/cn_img
  • attachment/photo
  • attachment/thumb
  • attachment/upload
  • attachment/mini
  • data
  • data/bbscache
  • data/groupdb
  • data/guestcache
  • data/style
  • data/tmp
  • data/tplcache
  • data/tplcache
  • data/forums
  • html
  • html/htm_data
  • html/channel
  • html/portal/bbsindex
  • html/portal/bbsindex/config.htm
  • html/portal/bbsradio
  • html/portal/bbsradio/config.htm
  • html/portal/oindex
  • html/portal/oindex/config.htm
  • html/portal/bbsindex/main.htm
  • html/portal/bbsindex/index.html
  • html/portal/bbsradio/main.htm
  • html/portal/bbsradio/index.html
  • html/portal/oindex/main.htm
  • html/portal/oindex/index.html

第三步：执行安装

请在浏览器中运行 install.php,即 http://您的论坛/install.php 安装程序，填入安装相关信息与资料，完成安装！使用论坛的用户名密码登陆后台进行管理和发布操作!!注意根据实际情况填入数据库信息和创始人信息。

本地安装 空间安装
数据库服务器 localhost 空间IP（localhost.or请咨询空间商）
数据库用户名 本地数据库用户名 空间数据库用户名（请咨询空间商）
数据库密码 本地数据库密码 空间数据库密码（请咨询空间商）
数据库名 phpwind（可以随意） 空间数据库名（请咨询空间商）
数据库类型 Mysql (若支持可以选择mysqli) Mysql (若支持可以选择mysqli)
